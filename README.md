# Student Management System

## Description
This is a simple Student Management System built using HTML, CSS, and JavaScript.

## Features
- Add student details
- Edit student information
- Delete student records
- Search for students

## Technologies Used
- HTML
- CSS
- JavaScript

## Installation
1. Download or clone the project.
2. Open the project folder.
3. Double-click `index.html` or open it in a web browser.

## Folder Structure
```
project/
│── index.html
│── style.css
│── script.js
│── README.md
```
